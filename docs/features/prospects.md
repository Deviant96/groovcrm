# Prospects

- Paginated data table with search, sort, filters, column resize/hide, multi-select, sticky/frozen columns
- Status pipeline: New → Sent → Replied → Interested → Meeting → Proposal → Closed Won / Closed Lost
- Detail page with inline edit (double-click, Enter/Esc)
- Notes (unlimited) and automatic activity timeline
- Follow-up date drives Home “Today” and “Overdue” lists
- Bulk status update and bulk delete
- Quick actions: WhatsApp, copy phone, open Instagram/website, add note, change status, schedule follow-up
