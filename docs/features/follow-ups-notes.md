# Follow-ups & notes

## Follow-ups

Each prospect may have `follow_up_date`. Home shows:

- Today's follow-ups
- Overdue follow-ups

Filters on the prospects table also support today / overdue / upcoming / none (labels: “Follow-up today”, “Overdue follow-up”, etc.).

## Notes

Unlimited notes per prospect, shown on the detail timeline with activities. Summary `notes` field remains available for a short blurb on the prospect record.
